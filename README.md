# GPSoftwareServices Support Manager

WordPress plugin for managing customers, devices, maintenance contracts, and technical support interventions.

## Development

The complete source code of the plugin is publicly available in this repository.

The distributed JavaScript file:

`assets/js/admin-v2.js`

is intentionally shipped in human-readable form. A corresponding development source copy is available at:

`assets/src/admin-v2.source.js`

No build step is required. Both files contain the same application logic.
