# gpsoftwareservices-support-manager
WordPress plugin for managing customers, devices, maintenance contracts and technical support interventions.
## Development

The complete source code of the plugin is available in this repository.

The distributed JavaScript file:

`assets/js/admin-v2.js`

has its human-readable source available here:

`assets/src/admin-v2.source.js`

No build step is required. The distributed JavaScript is functionally equivalent to the readable source file.
